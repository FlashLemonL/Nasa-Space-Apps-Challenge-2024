A game made for the Nasa Space Apps Challenge 2024.

Made to educate on some aspects of living in a space station.

A Dependency is pygame.
To install for windows:    
Enter     
py -m pip install -U pygame --user   
Into your command prompt

For MAC  
Enter  
python3 -m pip install -U pygame --user  
Into you terminal.

For more info on how to install pygame go to the website by clicking <a href=https://www.pygame.org/wiki/GettingStarted> this</a>
